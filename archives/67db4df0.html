Java

1.JAVA_HOME在变量值窗口中选择浏览目录，找到你jdk所在的文件夹（不是bin目录，是bin目录所在的上一级！！！）

2.%JAVA_HOME%\bin

Maven

1.新建变量MAVEN_HOME = E:\Tools\Maven\apache-maven-3.8.1（以自己的安装路径为准）

2.编辑变量Path，添加变量值%MAVEN_HOME%\bin

3.win+R运行cmd，输入

mvn help:system

4.附件

Tomcat

1.在系统变量中新建

CATALINA_HOME
变量值为Tomcat的安装目录。

2.在Path的变量值中加入：

%CATALINA_HOME%\bin

%CATALINA_HOME%\lib

Redis

进入redis根目录，输入下面的指令启动服务器：

redis-server.exe redis.windows.conf
